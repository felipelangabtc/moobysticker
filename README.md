# MoobyStickers

NFT Album for Mooby Stickers collection.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Wagmi (Web3)
- Supabase

## Getting Started

```bash
# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev
```

## Features

- NFT gallery and album
- Web3 integration with Polygon
- Supabase backend
